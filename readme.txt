=== LISL Last-Image Slider ===
Tags: image, slider, theme, nivo, timthumb, photo

 Last-Image Slider, is an implementation of Nivo slider (http://nivo.dev7studios.com), using TimThumb resizer (http://www.darrenhoyt.com/2008/04/02/timthumb-php-script-released/ ), to display last uploaded images with a jquery cool slider effect.

== Installation ==

1. Activate the plugin
2. Go to LISL option page under Themes
3. Configure how many images would you like to see in the slider
4. Configure max images size
5. Save your configuration
6. Now you can edit your theme and put the code to activate the slider.
